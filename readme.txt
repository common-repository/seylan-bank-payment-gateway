== Description ==

The Seylan Bank Payment Gateway Plugin allows you to incorporate Seylan Bankâ€™s online payment gateway to your WordPress e-commerce website that uses WooCommerce. Once the plugin is installed, customers can use major debit and credit cards to complete their transactions.  

The Seylan Bank Payment Gateway Plugin processes the debit or credit card payments for online transactions. It facilitates the transactions made by customers by sending key information between the e-commerce website and Seylan Bank.

The bank payment gateway authorizing the payment between the customer and your online shop. When the customer completes a transaction from your WooCommerce site, the payment gateway encrypts the data and sends it to the Seylan Bank. The bank views the request and will either approve or deny it.Once the bank authorizes the transaction, it transmits the payment. All of these steps take only a couple of seconds to complete. 

The advantage of using the Seylan Bank Payment Gateway Plugin is that customers can do their transactions with your online shop without any hassles.It secures the transaction to ensure that it is safe from hackers and other third party. 

The Seylan Bank Payment Gateway Plugin is user-friendly, easy to setup, and frees to download and use.


== Installation ==

**Automatic installation**

Automatic installation is the easiest option as WordPress handles the file transfers 
itself and you don't need to leave your web browser. To do an automatic install, 
log in to your WordPress dashboard, navigate to the Plugins menu and click Add New.

In the search field type plugin name and click Search Plugins. Once you've found 
our plugin you can install it by simply clicking Install Now.

**Manual installation**

1.Download plugin zip file
2.Unzip and upload to (wp-content/plugins) directory or upload zip file and install it 
   from Plugins->Add new->Upload
3.Activate the plugin through the Plugins menu in WordPress.
4.Enter payment gateway details in woocommerce->settings->checkout->[IPG NAME] and Enable payment gateway
5.That's all you need to do. You will notice Payment gateway option in Checkout page.
